## 🇬🇧 English

### About
This repository contains **311 PDF textbooks and teacher guides** for the Lao national curriculum, organized by level, grade, and subject. All files are sourced from official Lao educational materials.

### Contents

| Category | Grades | Files |
|---|---|---|
| 📘 Textbooks – Primary | ປ1 – ປ5 (Grades 1–5) | 58 |
| 📗 Textbooks – Secondary | ມ1 – ມ7 (Grades 6–12) | 112 |
| 📒 Teacher Guides – Primary | ປ1 – ປ5 | 57 |
| 📓 Teacher Guides – Secondary | ມ1 – ມ7 | 116 |

### File Naming Convention
`[grade]_[subject].pdf`
Examples: `p1_mathematics.pdf`, `m3_lao-language.pdf`, `m5_physics.pdf`
New-edition textbooks include year: `p1_mathematics_2020.pdf`
